## 分页参数说明

| 参数     | 默认值 | 解释         | 特殊                                                         | 请求可选 |请求参数|响应参数|
| -------- | ------ | ------------ | ------------------------------------------------------------ | ---- |----|----|
| offset  | 1      | 页码         | offset < 1，返回第一页 ; offset > 有效页数 ，data 为空     | 是   |是|是|
| limit | 10     | 每页数据条数 | limit = 0，返回全部 (会忽略 pageNum 参数) ; page < 0 , data 为空 | 是   |是|是|
